# Six main point about GO

1. Statically typed language
var x int = 10

2. Strongly typed language
You can't assign a value of one type to a variable of another type

3. Go is compiled language

4. Fast compile time

5. Built-in concurrency

6. Simplicity

## Run the go file
```bash
go run main.go
```

## To build and run
```bash
go build main.go
./main
```
